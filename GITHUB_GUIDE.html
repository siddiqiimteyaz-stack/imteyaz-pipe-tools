GITHUB UPLOAD GUIDE

IMTEYAZ PIPE TOOLS — 10 minute setup · hamesha ke liye free
STEP 1
GitHub par free account banao

Apne phone ya laptop mein browser kholo aur jaao:

👉 github.com

"Sign up" par click karo → email, username, password dalo → account ban jaayega.

✅ Pehle se account hai? Seedha Step 2 par jaao.
STEP 2
New Repository banao

Login karne ke baad upar "+" icon par click karo → "New repository" select karo.

Yeh fields bharo:

• Repository name: imteyaz-pipe-tools

• Public select karo (zaroori hai)

• "Add a README file" ka checkmark lagao

Phir green button "Create repository" dabao.

STEP 3
HTML file upload karo

Repository page par "Add file" → "Upload files" par click karo.

Apne phone/laptop se imteyaz_pipe_tools.html file choose karo.

File upload hone ke baad neeche "Commit changes" green button dabao.

⚠ File ka naam exactly yahi rakhna: index.html — upload se pehle rename kar lo!
STEP 4
GitHub Pages ON karo

Repository mein upar "Settings" tab par click karo.

Left sidebar mein "Pages" par click karo.

"Source" mein "Deploy from a branch" select karo.

"Branch" mein "main" select karo → "Save" dabao.

⏳ 2-3 minute wait karo — phir aapka app live ho jaayega!
STEP 5
App ka link milega

Settings → Pages page par wapas jaao — wahan aapka link dikhega:

https://[aapka-username].github.io/imteyaz-pipe-tools

Yeh link kisi ko bhi bhejo — WhatsApp par, site engineers ko, fabricators ko.

STEP 6
Phone mein Install karo

Apne Android phone mein Chrome se woh link kholo.

Screen par neeche "INSTALL APP" button aayega — dabao.

Ya Chrome ke 3 dots → "Add to Home Screen"

App home screen par install ho jaayega — bilkul Play Store jaisa! ✅

✅ HO GAYA!

Aapka app ab poori duniya mein available hai — free mein, hamesha ke liye.

Koi bhi link se install kar sakta hai — offline bhi chalega.
